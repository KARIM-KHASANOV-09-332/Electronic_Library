# Tests

Тесты запускаются без реальной базы данных: обращения к PostgreSQL заменяются на тестовый `FakeDatabase`.

Команда для запуска из backend-контейнера:

```bash
docker compose up -d --no-build db backend worker frontend
docker compose exec backend python -m unittest discover -s tests -p "test_*.py" -t . -v
```

Что покрыто:

- валидация пользователя, телефона, пароля и читательского билета;
- регистрация читателя и автора;
- вход с неверными данными;
- смена ролей в админ-панели;
- выдача книги читателю;
- запрет выдачи книги не-читателю;
- понятное сообщение при отсутствующем файле книги;
- создание закладки;
- модерация книги;
- наличие ключевых таблиц и миграций;
- наличие основных frontend-страниц и кнопок для демонстрации.
